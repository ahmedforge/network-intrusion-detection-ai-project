## Requirements
Install dependencies:
\```
pip install -r requirements.txt
\```
## How to Run
1. Place `network_traffic.csv` in the same folder as the notebook
2. Open `NIDS_Project.ipynb` in Jupyter Notebook or JupyterLab
3. Run all cells top to bottom (Kernel → Restart & Run All)

## Output
All plots are saved automatically as `.png` files in the same folder.
